CSS Selectors

Apllying CSS to Different Parts of HTML


  1. The element selector targets elements based on their HTML tag name.
  2. Class selector target elements based on the value of the id attribute.
  3. ID selectors target elements based on the value of the id attribute.
  4. Attribute selectors target elements based on their attributes and values.
  5. The universal selector targets all elements.

Element selectors target elements based on their HTML tag name, while class selectors target elements based on the value of the class attribute.

ID selectors focus on elements based on the value of the id attribute, and attribute selectors target elements based on their attributes and specific values. The universal selector targets all elements on the page.
